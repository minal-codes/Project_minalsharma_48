# Smart Vehicle Maintenance Tracker

A C-based application to track fleet vehicle maintenance, predict next service, and send overdue alerts.

## Features
- Store vehicle details using structures
- Use functions & arrays for operations
- Predict next service based on usage
- Alert overdue vehicles
- Dynamic memory allocation using pointers
- Fully modular design

## How to Compile
```bash
gcc src/smart_vehicle_tracker.c -o tracker
./tracker
